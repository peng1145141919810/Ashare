# 接入步骤（教学版）

## 1. 先改配置
打开：

- `configs/hub_config.v5.local.json`

重点检查：

- `train_table_dir`
- `hub_output_root`
- `execution.python_executable`

你的 Python 路径就用：

```json
"python_executable": "C:/Users/Administrator/PyCharmMiscProject/.venv/Scripts/python.exe"
```

## 2. 再改本地模式
打开：

- `hub/local_settings.py`

### 先校验环境
```python
CONFIG_PATH = r"configs/hub_config.v5.local.json"
MODE = "validate_only"
DRY_RUN = True
MAX_CYCLES = 1
SLEEP_SECONDS = 0
```

### 再只看计划
```python
MODE = "plan"
DRY_RUN = True
MAX_CYCLES = 1
```

### 再跑一轮真实实验
```python
MODE = "batch"
DRY_RUN = False
MAX_CYCLES = 1
```

### 最后进入持续闭环
```python
MODE = "adaptive_research_brain"
DRY_RUN = False
MAX_CYCLES = None
SLEEP_SECONDS = 0
```

## 3. PyCharm 里点运行
直接运行：

- `run_research_hub_v5_local.py`

不要再开 PowerShell。

## 4. 跑完先看哪些文件

### 总控状态
- `hub_output_root/controller_state.json`

### 总账本
- `hub_output_root/registry/experiment_registry.csv`

### 最近一轮
- `hub_output_root/cycles/<cycle_id>/cycle_plan.json`
- `hub_output_root/cycles/<cycle_id>/cycle_summary.json`
- `hub_output_root/cycles/<cycle_id>/cycle_manifest.csv`

### 策略家族
- `hub_output_root/strategy_family/strategy_family_state.csv`
- `hub_output_root/strategy_family/deployment_gate.json`

## 5. LLM 怎么接
配置里填：

```json
"llm_brain": {
  "enabled": true,
  "base_url": "https://api.openai.com/v1",
  "api_key_env": "OPENAI_API_KEY",
  "model": "gpt-5.4-thinking",
  "temperature": 0.2,
  "timeout_seconds": 120
}
```

注意：

- `api_key_env` 写的是 **环境变量名**，不是密钥本身。
- 如果要接 DeepSeek，把 `base_url` 和 `api_key_env` 改成对应值即可。

## 6. 为什么 V5 不会一秒退出
因为 V5 的候选工厂：

- 冷启动自带父代种子
- 同轮去重，跨轮不去重
- 至少按预算产出候选

所以只要数据能读、配置没坏，V5 冷启动就应该真正开始执行候选实验。
